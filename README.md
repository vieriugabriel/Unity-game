# Physics-Based 2D Hill Climb Racer 🏎️🏔️

A complete hyper-casual mobile racing game built with **Unity** and **C#**, focusing on realistic vehicle physics and cloud-based player data.

## 🌟 Key Features
* **Custom 2D Physics:** Realistic suspension, torque, and tire-to-ground interaction for a smooth driving feel.
* **Firebase Integration:** Secure user authentication (Email/Google) and real-time cloud data synchronization.
* **Global Leaderboards:** Competitive scoring system using Firebase Realtime Database.
* **Mobile Optimized:** 60 FPS performance on mid-range Android devices with optimized sprite management.

## 🛠️ Tech Stack
* **Engine:** Unity 2D.
* **Language:** C#.
* **Backend:** Firebase (Auth, Database, Cloud Save).
* **Target Platform:** Android (supports SDK 21+).

## 🚀 How to Play
1. Clone the repository and open the project in **Unity 2021.3+**.
2. Connect your own **Firebase GoogleService-Info.plist** / **google-services.json**.
3. Build and run on an Android device or emulator.
